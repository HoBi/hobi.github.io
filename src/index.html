Know me...

I am 18 years old independent coder and programmer. I live in Ceska Lipa which is a city situated in the north of the Czech Republic. I am interested in web technologies, which I also study. In my free time I study languages like HTML5, CSS3 with the Sass preprocessor - and OOCSS with BEM syntax, JavaScript with the jQuery library, Dart and PHP.

I am a member of Nouvelle Communauté which is a student community mainly focused on IT. Nouvelle Communauté is a brand that represents our collective strength, skill and inventiveness.

I attend an IT class in a technical school, where we learn Pascal, C#, Java and a bit of Assembler. For my projects I use versioning control system Git for which I use the GitHub repositores.

I also write some articles about IT stuff, mostly about coding, on a programming social network called ITnetwork.cz. I am also an editor and a moderator of this website.
